# Exam 01: HTML Basics

Use this as a short concept check after building.

## Multiple Choice

1. What does `<h1>` usually create?
   - A. A text box
   - B. A main heading
   - C. A button
   - D. A browser tab name

2. What does `<p>` do?
   - A. Creates a paragraph
   - B. Creates a picture
   - C. Saves the file
   - D. Connects CSS

3. What does `<button>` create?
   - A. A heading
   - B. A clickable button
   - C. A folder
   - D. A browser tab

4. What does `<input>` do?
   - A. Creates a place for the user to type
   - B. Creates a heading
   - C. Creates a section title
   - D. Plays music

5. What is a `<div>` mostly used for?
   - A. Making text bold
   - B. Grouping things together
   - C. Opening GitHub
   - D. Renaming files

6. Why do we indent code?
   - A. To make it easier to read
   - B. To make the site load faster
   - C. To publish to GitHub
   - D. To turn HTML into CSS

7. Why might we leave a blank line in code?
   - A. To organize sections and make code easier to read
   - B. To save the project
   - C. To add JavaScript
   - D. To install extensions

8. What does `<!DOCTYPE html>` tell the browser?
   - A. This is a CSS file
   - B. This is an HTML page
   - C. Open GitHub
   - D. Add a button

9. What is `<h2>` usually used for?
   - A. A section heading
   - B. A text box
   - C. A browser tab name
   - D. A saved song

10. What does CSS mainly control?
   - A. How the page looks
   - B. How the file saves
   - C. How GitHub works
   - D. How buttons think
